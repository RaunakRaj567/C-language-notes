## 🎯 What Each Unit Covers

### 📚 Unit 1: C Language Basics
- Character set, Identifiers, Keywords
- Data types (int, float, char, double, void)
- Variables, Constants, Scope
- Input/Output (printf, scanf)
- Operators, Expressions
- **30 Viva Questions** on basics

### ⚡ Unit 2: Control Structures
- if, else, else-if statements
- switch, case, break
- while, do-while, for loops
- continue statement
- Logical & Relational operators
- **30 Viva Questions** on control flow

### 🔧 Unit 3: Functions & Storage Classes
- Function declaration, definition, calling
- Return types and parameters
- Pass by value vs reference
- Recursion
- Scope (local, global)
- Storage classes (auto, static, extern, register)
- **30 Viva Questions** on functions

### 📦 Unit 4: Arrays
- 1D Array declaration and initialization
- 2D Arrays (matrices)
- Array indexing and traversal
- Strings as character arrays
- Linear and Binary search
- Bubble sort, Selection sort
- **30 Viva Questions** on arrays

### 🎯 Unit 5: Pointers & Strings
- Pointer basics (& and * operators)
- Pointer declaration and dereferencing
- Pointer arithmetic
- Double pointers
- Dynamic memory (malloc, calloc, free)
- String functions (strlen, strcpy, strcat, strcmp)
- **30 Viva Questions** on pointers

### 🏗️ Unit 6: Structures & Unions
- Structure declaration and initialization
- Accessing structure members
- Nested structures
- Array of structures
- Unions and shared memory
- Bit fields
- Self-referential structures (linked lists)
- **30 Viva Questions** on structures

### ⭐ Unit 7: C++ Basics
- Classes and objects
- Constructors and destructors
- Inheritance and polymorphism
- Virtual functions
- Exception handling (try-catch)
- STL containers (vector, map)
- String class
- Templates
- Smart pointers
- **30 Viva Questions** on C++
