## 🚨 Troubleshooting

**Issue:** Port 3000 in use
```bash
PORT=3001 npm start
```

**Issue:** Dependencies error
```bash
rm -rf node_modules
npm install
```

**Issue:** Hot reload not working
```
Press Ctrl+Shift+R (hard refresh)
```
