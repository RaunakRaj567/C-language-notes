## 📚 Course Structure

### Unit 1: 📚 C Language Basics
- Character set, Identifiers, Keywords
- Data types, Variables, Constants
- Input/Output Functions (printf, scanf)
- Operators, Expressions

### Unit 2: ⚡ Control Structures
- if, else, switch statements
- Loops (while, do-while, for)
- Break, Continue, Goto
- Logical & Relational Operators

### Unit 3: 🔧 User Defined Functions & Storage Classes
- Function declaration, definition, calling
- Pass by value & reference
- Recursion
- Scope, Static, Extern, Register

### Unit 4: 📦 Arrays in C
- 1D and 2D Arrays
- String handling
- Searching (Linear, Binary)
- Sorting (Bubble, Selection)

### Unit 5: 🎯 Pointers & Strings
- Pointer basics, & and * operators
- Pointer arithmetic
- Dynamic memory (malloc, calloc, free)
- String functions (strlen, strcpy, strcat, strcmp)

### Unit 6: 🏗️ Structures & Unions
- Structure declaration & initialization
- Nested structures, Array of structures
- Unions, Bit fields
- Self-referential structures

### Unit 7: ⭐ C++ Basics
- Classes & Objects
- Constructors & Destructors
- Inheritance & Polymorphism
- Exception Handling, STL, Templates

## 🚀 Setup और Run करने के लिए

### Requirements
- Node.js (v14 या उससे ऊपर)
- npm या yarn

### Installation Steps

```bash
# 1. Project directory में जाओ
cd c-learning-portal

# 2. Dependencies install करो
npm install

# 3. Development server start करो
npm start

# 4. Browser में खुल जाएगा: http://localhost:3000
```

### Build करने के लिए (Production)

```bash
npm run build
# यो एक optimized build create करेगा जो deploy कर सकते हो
```
